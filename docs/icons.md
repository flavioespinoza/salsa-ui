# Icons

All icon components are exported as React components with customizable `className`.

## Usage

```tsx
import {
	AirdropsIcon,
	DocumentationIcon,
	HomeIcon,
	LogoIcon,
	LogoutIcon,
	MenuIcon,
	MyNodesIcon,
	OrchestrationIcon,
	ReferralsIcon,
	SettingsIcon,
	StakingIcon
} from '@flavioespinoza/salsa-ui'

<HomeIcon className="text-gray-600 h-6 w-6" />
```
